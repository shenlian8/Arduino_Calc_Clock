# Arduino LED矩阵计算时钟

这是一个基于Arduino的创意时钟项目。它使用一个10段8x8 LED点阵显示器来展示时间，并且拥有两种独特的显示模式。

## 功能特点

- **标准时间显示**：以 `HH:MM` 和 `MM-DD` 的格式清晰地显示当前时间和日期。
- **趣味算式显示**：将当前的小时和分钟转换成一个结果等于该数字的随机数学算式（例如，时间`14:35` 可能会显示为 `7+7` 和 `5X7`）。
- **模式切换**：通过一个物理按钮，可以轻松地在“标准时间”和“算式时间”两种模式之间切换。
- **高精度计时**：集成DS3231实时时钟(RTC)模块，确保即使在断电后也能保持准确的时间。
- **自动时间设置**：首次启动时，时钟会自动设置为代码编译的日期和时间。
- **生日彩蛋**：代码中预设了一个生日（10月26日），在当天会显示特别的生日祝福。

## 所需硬件

1.  **Arduino主板**：例如 Arduino Uno, Nano 或任何兼容的板子。
2.  **MAX7219/MAX7221 LED点阵模块**：一个由10个8x8点阵组成的串联模块。
3.  **DS3231 实时时钟 (RTC) 模块**：用于精确计时。
4.  **瞬时按钮**：一个用于切换显示模式的按钮。
5.  **杜邦线和面包板**

## 所需库

请在Arduino IDE的库管理器中安装以下库：

1.  `Rtc by Makuna` (用于DS3231)
2.  `LEDMatrixDriver`

## 接线说明

### 1. 电源连接 (Power Supply)

*   **Arduino 5V** -> 连接到面包板的 **正极（+）** 电源轨。
*   **Arduino GND** -> 连接到面包板的 **负极（-）** 电源轨。

### 2. LED 点阵模块 (MAX7219)

*   **VCC** -> 面包板 **正极（+）**
*   **GND** -> 面包板 **负极（-）**
*   **DIN** (Data In) -> Arduino **D11**
*   **CS** (Chip Select) / **LOAD** -> Arduino **D9**
*   **CLK** (Clock) -> Arduino **D13**

### 3. 实时时钟模块 (DS3231)

*   **VCC** -> 面包板 **正极（+）**
*   **GND** -> 面包板 **负极（-）**
*   **SCL** (Serial Clock) -> Arduino **A5**
*   **SDA** (Serial Data) -> Arduino **A4**

### 4. 按钮 (Push Button)

*   将按钮的一个引脚连接到 Arduino **D2**。
*   将按钮的另一个引脚连接到面包板 **负极（-）**。

## 使用说明

1.  按照上面的“接线说明”连接好所有硬件。
2.  在Arduino IDE中打开 `.ino` 文件。
3.  安装所需的库。
4.  （可选）在代码中修改 `birthMonth` 和 `birthDay` 变量为你自己的生日。
5.  将代码编译并上传到你的Arduino主板。
6.  时钟启动后，会默认显示“CLOCKINIT.”，然后开始显示时间。
7.  按下连接到D2引脚的按钮，可以在两种不同的时间显示模式之间循环切换。
